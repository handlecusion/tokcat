Tokcat

Your AI coding token usage, alive in the macOS menu bar.

macOS 11+ Apple Silicon Tauri 2 MIT No telemetry

Tokcat is a native macOS menu-bar app, built with Tauri 2 (Rust + React/Vite), that visualizes your AI coding token spend across Claude Code, OpenAI Codex, Cursor, OpenCode, Google Gemini, GitHub Copilot, Amp, and Droid as a 2D or 3D GitHub-style contribution graph. The animated cat icon shows today's tokens or USD cost; clicking opens a frosted-glass popover with per-client filters, streak summaries, and a settings panel. Tokcat refreshes every 3 minutes by shelling out to tokscale graph --no-spinner, checks for signed updates every 30 minutes, ships zero telemetry, and requires no account.

Install

brew tap handlecusion/tokcat
brew install --cask tokcat

Or grab the latest signed DMG from GitHub Releases.

FAQ

What is Tokcat?

Tokcat is a free, open-source native macOS menu-bar app that visualizes your AI coding token usage as a 2D or 3D GitHub-style contribution graph. It wraps the tokscale CLI in a Tauri 2 shell and surfaces sessions from Claude Code, Codex, Cursor, OpenCode, Gemini, Copilot, Amp, and Droid in one glanceable place. The app runs entirely on-device, makes zero analytics requests, and requires no account.

How much does Tokcat cost?

Tokcat is free and open-source under the MIT licence. There is no subscription, no paid tier, and no telemetry. Install with brew tap handlecusion/tokcat && brew install --cask tokcat.

Which AI coding tools does Tokcat track?

Tokcat tracks every AI coding client that tokscale supports: Claude Code, OpenAI Codex, Cursor, OpenCode, Google Gemini, GitHub Copilot, Amp, and Droid. Because Tokcat shells out to tokscale graph --no-spinner every 3 minutes, any new client added upstream appears automatically.

Does Tokcat send my data anywhere?

No. Tokcat's only network request is a check against the GitHub Releases updater manifest. There is no telemetry, no analytics, no cloud sync, no account, and no third-party server. All token data is read locally by tokscale.

How is Tokcat different from tokscale on its own?

tokscale is a command-line tool. Tokcat is a native macOS GUI on top of it — an animated menu-bar cat, a click-to-open frosted-glass dashboard with a 2D heatmap and an interactive 3D tile graph, per-client filters, streaks, and a settings panel.

Does Tokcat run on Intel Macs or Windows?

Tokcat ships only for Apple Silicon (arm64) on macOS 11+. There is no Intel x86_64 build and no Windows or Linux build. For cross-platform usage, use tokscale directly.

Privacy

No telemetry. No cloud sync. No account. Tokcat's only network traffic is the updater manifest fetched from github.com/handlecusion/tokcat/releases/latest/download/latest.json every 30 minutes. All token-usage data is read locally from session logs by the bundled tokscale CLI.
